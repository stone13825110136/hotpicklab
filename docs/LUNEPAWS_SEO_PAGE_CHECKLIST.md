## 全站统一规则

1. **Title**：主关键词靠前 + 情境词（friend / first week / printable）

3. **每个指南至少 3 条 FAQ**，问题尽量等于真实搜索句
4. **工具页**用「free / printable / generator」词，比商品词更容易起量
